`add` plugin adds custom text to your output file.

You can use this plugin to add custom code, imports, comments and more to your output file.

## Examples

```yaml {5-6}
# …
generates:
  path/to/file.ts:
    plugins:
      - add:
          content: '/* eslint-disable */'
      - typescript
```

```yaml {5-10}
# …
generates:
  path/to/file.ts:
    plugins:
      - add:
          content:
            - 'declare namespace GraphQL {'
      - add:
          placement: 'append'
          content: '}'
      - typescript
```
