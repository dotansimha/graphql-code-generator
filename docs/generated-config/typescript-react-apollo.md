
### withComponent (`boolean`, default value: `true`)

Customized the output by enabling/disabling the generated Component.


#### Usage Example

```yml
generates:
path/to/file.ts:
 plugins:
   - typescript
   - typescript-operations
   - typescript-react-apollo
 config:
   withComponent: false
```
